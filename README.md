# App Discos - Web

Aplicación web desarrollada en ASP.NET WebForms como práctica, 
basada en una aplicación de escritorio previa.

## Tecnologías utilizadas
- C#
- ASP.NET WebForms
- SQL Server
- ADO.NET

## Funcionalidades
- Listado de discos
- Alta de nuevos discos
- Arquitectura en capas (Dominio, Negocio, Web)

## Objetivo
Proyecto realizado con fines educativos y de práctica
en los cursos de maxiprograma.com,
para afianzar conocimientos en desarrollo web con .NET.
